Installation
------------

git clone https://bitbucket.org/hotjar/dev-task-jeremy.git
cd dev-task-jeremy
cp env.bash.example env.bash
vim env.bash   <-- edit if required

cd app
pip3 install -r requirements.txt


Quickstart
----------

To run the app:

cd app
apistar run


Testing
-------

To test the app:

cd app
apistar test
